# SASH-VPV Subcutaneous Vascular Palm Vein Authentication Model

This repository builds and deploys a Deep Learning pipeline focused on Biometric verification using the unique subcutaneous human palm vascular networks found within the **SASH-VPV Dataset**.

## 🚀 How to Deploy on Streamlit Cloud
1. Push this complete project repository structure to your GitHub.
2. Sign in to your **Streamlit Community Cloud** account.
3. Click `New app` and select your pushed repository.
4. Set **Main file path** to `streamlit_app.py`.
5. Click **Deploy!** The `kagglehub` API downloads the raw assets into the background dynamically.
